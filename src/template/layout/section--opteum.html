Приложение для водителей(таксометр)

Поддержка Android, история заказов, расчет по таксометру, навигатор.

Настройка прав и ограниченние доступа

Возможность создавать группы пользователей и назначать права.

Учет и отчетность

Сохранение всей истории по заказам, формирование отчетности по водителям, контрагентам.

Рабочее место диспетчера

Работа с картой, быстрый ввод адресов, архив заказов, черный список.

Автоматический расчет стоимости

Возможность расчета стоимости по координатам. Загрузка адресов города.

GPS-мониторинг транспорта

Контроль местоположения экипажей такси, сохранение трека по заказу.

Финансовые расчеты

Автоматические расчеты с водителем, диспетчером, контрагентом. Работа с лицевым счетом.

Персональный таксометр

Создание собственных тарифов, учет ожидания, подачи, минималка.

Автоматическое распределение заказа

Распределение по ближайшему водителю и по парковочным зонам.

Клиентская база и черный список

Ведение истории по клиенту, отказы и черный список по клиентам такси.

Сall-центр

Привязка СИП-номера, входящие исходящие звонки, автоматические отзвоны ожидание, голосовое меню.
